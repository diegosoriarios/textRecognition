# Text Recognition
Faz o reconhecimento de texto em imagens utilizando [Tesseract.js](http://tesseract.projectnaptha.com/)

### Instalation

```
    git clone https://github.com/diegosoriarios/textRecognition.git && cd textRecognition
    npm install
    node index.js
```
